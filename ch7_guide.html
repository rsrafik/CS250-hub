üïíClocking + Setup/Hold

üó∫Ô∏èFinite-State Machines (FSM)
FSM = state register + next-state function + output function
If state storage has n bits ‚Üí 2n states
Next-state is combinational: (state, inputs) ‚Üí next state
Outputs come from state (and maybe inputs)
Moore: outputs depend only on state (often faster).
Mealy: outputs depend on state + inputs (often fewer states).
NextState = f(CurrentState, Inputs) Outputs = g(CurrentState) // Moore Outputs = g(CurrentState, Inputs) // Mealy
Implementation: state register holds CurrentState; combinational logic computes NextState + outputs each cycle.
